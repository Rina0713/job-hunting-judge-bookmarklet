# 🔍 求人情報一括抽出・多段階自動判定ブックマークレット

## 💻 概要
本ツールは、人材紹介業務における求人選定および精査プロセスを効率化するために開発した、JavaScript製のブラウザ用スクリプト（ブックマークレット）です。
求人一覧画面から、バックグラウンド処理にて各求人の詳細データを高速で取得・解析し、候補者の希望条件や除外条件（シフト制の有無、転勤の有無など）との合致度を5段階のスコアで算出後、別ウィンドウへレポートとして一覧出力します。

## 🎯 開発の背景と解決した課題
従来のエージェント業務では、求人一覧から1件ずつ詳細ページを新規タブで開き、「シフト制か」「転勤があるか」「指定キーワードが含まれるか」を目視で確認する必要があり、候補者1名あたり100件規模の精査を行う際に多大な工数が発生していました。
この現場におけるボトルネックを解消するため、「外部サーバーや専用環境を構築せず、ブラウザの標準機能だけでワンクリックで動作する」という要件のもと、独学で本ツールを構築しました。導入により、選定作業に要する時間を大幅に削減し、業務効率化を実現しました。

## ✨ 技術的なアプローチ・実装ポイント
- **非同期処理の制御とWebサーバー負荷軽減：** `fetch` APIを用いた詳細ページ（HTML）の連続取得処理を実装。スクレイピングに伴うブラウザのハングアップ防止や、相手方サーバーへの過度な負荷を避けるため、`Promise`と`setTimeout`を組み合わせ、ループ内に「0.05秒のディレイ（ウェイト）制御」を組み込んでいます。
- **DOM解析と動的フィルタリング：** 取得したレスポンスを`DOMParser`でパースし、テキスト内から特定のNG条件（「シフト休」「交代制」「転勤あり」など）を条件分岐で検知。合致した案件を瞬時に足切り（フィルタリング）するロジックを構築しました。
- **ユーザー体験（UX）への配慮：** 解析の待ち時間におけるユーザーの離脱や不安を防ぐため、進捗状況（解析完了件数 / 総件数 / マッチング数）をDOM操作によってリアルタイムで画面に動的反映させる仕様としています。
- **実務に即した多段階スコアリング：** 単なるキーワードの有無ではなく、指定キーワードの含有比率から5段階の適合度スコアを算出。さらに「社内カレンダーによる土曜出社」などの例外的な注記を自動検知して注意を促す判定ロジックを搭載しています。
